# 感情共有掲示板アプリ - 設計ドキュメント

## プロジェクト概要

**アプリ名:** Emotion Board（感情共有掲示板）  
**プラットフォーム:** Flutter (Android Studio)  
**目的:** 人から言われてうれしかったこと、つらかったことを匿名で共有できる掲示板

---

## 主要機能

### 1. 感情カテゴリー（下部タブ）
- **表情アイコンで選択**
  - 😊 うれしかったこと
  - 😢 悲しかったこと
  - 😫 つらかったこと
  - 😠 怒りを覚えたこと

### 2. 掲示板カテゴリー（上部スライドタブ）
- 仕事
- 家庭
- 学校
- その他

### 3. 5W1H入力フォーム
投稿時に以下を入力（「何を」以外はフィルター検索可能）：
- **いつ（When）**: 日時選択
- **どこで（Where）**: テキスト入力
- **誰から（Who）**: テキスト入力
- **どのカテゴリー（Category）**: 上記の感情カテゴリー
- **何を（What）**: フリーテキスト入力（フィルター不可）
- **どんなふうに（How）**: テキスト入力

### 4. フィルター検索
- いつ（日付範囲）
- どこで（キーワード）
- 誰から（キーワード）
- どんなふうに（キーワード）

### 5. ユーザー管理
- **匿名投稿**
- **初回登録時に入力**
  - 年齢（選択式：10代、20代、30代、40代、50代以上）
  - 性別（選択式：男性、女性、その他）

### 6. プロフィール画面
- 登録済みユーザー情報表示
- 投稿履歴表示

---

## データモデル

### User（ユーザー）
```
- id: String (UUID)
- age: String (10代, 20代, 30代, 40代, 50代以上)
- gender: String (男性, 女性, その他)
- createdAt: DateTime
```

### Post（投稿）
```
- id: String (UUID)
- userId: String (FK → User)
- boardCategory: String (仕事, 家庭, 学校, その他)
- emotionCategory: String (うれしい, 悲しい, つらい, 怒り)
- when: DateTime (いつ)
- where: String (どこで)
- who: String (誰から)
- what: String (何を - フリーテキスト)
- how: String (どんなふうに)
- createdAt: DateTime
- updatedAt: DateTime
```

## UI構成

### 画面一覧

1. **スプラッシュ/初期化画面**
   - ユーザー登録フォーム（年齢・性別）

2. **ホーム画面（メイン）**
   - 上部：掲示板カテゴリータブ（スライド可能）
   - 中央：投稿一覧（感情カテゴリーごとに表示）
   - 下部：感情カテゴリータブ（表情アイコン）

3. **投稿画面**
   - 5W1Hフォーム
   - 投稿ボタン

4. **詳細画面**
   - 投稿の詳細表示
   - 削除ボタン（自分の投稿のみ）

5. **フィルター検索画面**
   - 検索条件入力フォーム
   - 検索結果表示

6. **プロフィール画面**
   - ユーザー情報表示
   - 投稿履歴
   - 設定

## デザイン方針

- **色彩**: 感情を表現する色（うれしい→暖色、悲しい→冷色など）
- **タイポグラフィ**: 読みやすいフォント
- **アイコン**: 感情を表現する絵文字/アイコン
- **レイアウト**: モバイルファースト
- **アニメーション**: スムーズなトランジション

---

## 注意事項

- ユーザーは匿名だが、年齢・性別は登録される
- 投稿は削除可能（自分の投稿のみ）
- フィルター検索は「何を」以外で可能
- 掲示板カテゴリーと感情カテゴリーは独立している
